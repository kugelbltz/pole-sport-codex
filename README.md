# Pole Sport Codex
The **Pole Sport Codex** is an open, encyclopedia-style reference of competitive pole sport elements. It documents and organizes pole elements as defined by official code of points (such as IPSF, POSA, and others), providing a clear, structured way to explore move classifications, difficulty levels, and variations across federations.

The goal of the **Pole Sport Codex** is to serve athletes, coaches, judges, and organizers as a neutral, transparent resource for understanding how pole sport movements are defined.

Features / Scope:
- Structured move library based on official competition code of points
- Federation-agnostic design (supports multiple rule systems)
